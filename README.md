# Adaptive Hybrid LMS PLIC

## Overview
Verilog implementation of an Adaptive Variable Step-Size Hybrid LMS/Sign-LMS based Power Line Interference Canceller (PLIC) for ECG signal enhancement.

## Features
- Adaptive variable step-size filtering
- Hybrid LMS and Sign-LMS operation
- Dynamic mode switching
- Four-channel harmonic cancellation
- Fixed-point implementation
- Low-power VLSI architecture

## Repository Structure
- rtl/ : RTL source files
- testbench/ : Simulation environment
- constraints/ : FPGA timing constraints
- results/ : Error logs and performance metrics
- docs/ : Reports and figures

## Tools Used
- Verilog HDL
- Vivado 2024.1

## Target Applications
- Biomedical signal processing
- ECG denoising
- FPGA implementation
